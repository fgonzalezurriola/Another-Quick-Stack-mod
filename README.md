# Another Quick Stack Mod

A command that quickly stacks items into nearby chests and fills processing benches.

It does not require /op permissions → available in Adventure mode.

## Commands

`/quickstack` → Open settings UI (radius, inventory sections)

`/qs` → Fast stacking to nearby chests

`/qs fill` → Fill all nearby processing benches (furnaces, tanneries) with relevant items

`/qs fill furnace` → Fill only furnaces with ores and fuel

`/qs fill tannery` → Fill only tanneries with hides

## Configuration

The menu lets you toggle which inventory sections to check:
- **Storage** — Main inventory grid
- **Backpack** — Expandable backpack slots
- **Hotbar** — Quick access bar (off by default)
